Sistema de identificação e exibição do modelo de computadores

📌 Descrição

Este projeto é um sistema que identifica e exibe informações detalhadas sobre o modelo de um computador, processador, memória RAM, CPU, DiscoArmazenamento, Fonte.


🚀 Funcionalidades
Identificação  do modelo do computador
Exibição de informações de hardware (CPU, RAM, etc.)

🛠️ Tecnologias utilizadas
Java
